The first feature we will introduce is the alert. The container that holds the information in this box is called an alert. Use the following code for a sample alert:

<div class="callout secondary"> <h5>This is a secondary callout</h5> <p>It has an easy to override visual style, and is appropriately subdued.</p> <a href="#">It's dangerous to go alone, take this.</a> </div>

The second feature we will introduce is the button. Buttons offer interactivity and can be styled in a multitude of ways. Here is an example of a button:

Success

Here is a quickstart for buttons:

<a class="button primary" href="#">Primary</a> <a class="button secondary" href="#">Secondary</a> <a class="button success" href="#">Success</a> <a class="button alert" href="#">Alert</a> <a class="button warning" href="#">Warning</a>

This will produce:

Primary Secondary Success Alert Warning

The third and final feature we will introduce is the navigation bar, or the navbar. Navbars allow users quick and easy access to various parts of your site.

Here is a quickstart for buttons:

<div class="top-bar"> <div class="top-bar-left"> <ul class="dropdown menu" data-dropdown-menu> <li class="menu-text">Site Title</li> <li><a href="#">One</a></li> <li><a href="#">Two</a></li> <li><a href="#">Three</a></li> </ul> </div> <div class="top-bar-right"> <ul class="menu"> <li><input type="search" placeholder="Search"></li> <li><button type="button" class="button">Search</button></li> </ul> </div> </div>

This will produce:

  • Site Title
  • One
  • Two
  • Three
